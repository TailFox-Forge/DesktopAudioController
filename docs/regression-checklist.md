# DesktopAudioController 회귀 검증 체크리스트

## 목적

이 문서는 `v1.0.0` 직전과 이후 주요 릴리즈에서 반복 확인해야 할 핵심 시나리오를 고정합니다.

대상:

```text
- stable 후보 빌드
- v1.0.0 직전 최종 점검
- 장치/세션/트레이/업데이트 경로를 건드린 릴리즈
```

## 실행 전 공통 준비

```text
1. 테스트 대상 zip 압축 해제
2. 이전 실행 중인 DesktopAudioController 완전 종료
3. 필요 시 %LocalAppData%\DesktopAudioController\settings.json 백업
4. 로그 경로 비우거나 새 날짜 로그 기준선 확인
5. 테스트 대상 버전 기록
```

기록:

```text
테스트 버전:
테스트 일시:
테스트 환경:
테스터:
```

## A. 첫 실행 / 초기 진입

### A-1. 첫 실행 설정창 진입

```text
절차
1. 설정 파일이 없는 상태에서 앱 실행
2. 초기 설정창이 자동으로 뜨는지 확인
```


## E. 업데이트 경로

### E-1. 업데이트 안내 버튼 / 릴리즈 노트 버튼

```text

1. 새 버전이 있는 상태에서 앱 실행
2. 상단 업데이트 안내 문구 확인
3. "다운로드" 버튼 클릭

- 다운로드 버튼은 zip 자산 또는 릴리즈 페이지를 연다
- 릴리즈 노트 버튼은 GitHub 릴리즈 페이지를 연다
```
